AI Chat Widget Demo

Experience the power of AI-powered chat with customizable personalities

Features

  • Customizable AI personalities (Friendly Assistant, Professional Advisor, Tech Expert, etc.)
  • Powered by local Ollama models (Llama 3.1, Mistral, Phi-3, Gemma)
  • Responsive design inspired by iMessage and WhatsApp
  • Visitor behavior tracking and analytics
  • RAG-powered website knowledge (Supabase + vector embeddings)
  • Easy to embed in any website
  • 100% free hosting on Vercel

How to Use

Click the chat button in the bottom-right corner to start chatting with the AI assistant!

Installation

Add this to your website:

<!-- AI Chat Widget -->
<script src="https://your-domain.vercel.app/widget/ai-chat-widget.js"></script>
<ai-chat-widget
  api-url="https://your-domain.vercel.app"
  personality="customer-support"
  position="bottom-right"
  auto-open="false"
  tracking="true"
></ai-chat-widget>

Configuration Options

  • api-url: URL of your deployed chat application
  • personality: friendly-assistant, professional-advisor, tech-expert, customer-support, etc.
  • position: bottom-right, bottom-left, top-right, top-left
  • auto-open: true/false - automatically open widget after 2 seconds
  • tracking: true/false - enable visitor behavior tracking

Analytics

Access visitor analytics using JavaScript:

const analytics = window.AIChatWidget.getAnalytics();
console.log(analytics);
